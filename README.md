# TODO_APP

### My TODO_APP performs the following functionalities
* It creates a new todo.
* It can mark a todo as completed when we click on that todo item.
* It can change a completed item back to incomplete, indicating that it is to be done still(again).
* It has delete button that allows us to delete a todo item.
* It also stores and removes item to/from local storage.
* It also fetches items stored in local storage to populate our todo list when we reload or refresh our browser.
